# バフ

データベース

英語表記：Parallel
特性肩書：瞬間座標転移

奇襲や撤退に適したウェポンの一つ。
エンダーマンの力を借りることで座標を記憶することができるようになる。
最初の使用で自身がいる地点を記憶し、それ以降の使用では座標を記憶したうえで、過去の記憶先に使用者をテレポートさせる。
テレポートするとテレポート先に靄が現れる。
あえてエメラルドブロックへ着地するときに設置することで、テレポート時に即座にジャンプをできるようにするなど、常設ブロックとの相性も良い。
さらに、奈落に落ちながら使うことで、次に復帰する地点を過去のランダムな地点に変更することができる。
ただし、使うと転送先までの直線距離に応じた硬直がデバフとして付与される為、十分に注意して使わないとならない。
他のウェポンより長けている点は、如何なる時でも場所問わず身の安全を確保しつつ撤退ができるという点。

転送時硬直：(直線距離 x 0.75)秒 (最大3秒)
初使用時クールダウン：2秒
それ以降クールダウン：14秒


擬態モデル




ウェポン調整履歴

PHASE-02

・テレポート時の透明化を削除 (弱体化)
・奈落に落ちないセーフティーテレポートを追加 (強化)

PHASE-03

・テレポート時の距離に比例する硬直デバフの追加 (弱体化)
