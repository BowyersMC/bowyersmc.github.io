ウェポンの紹介

通常ウェポン

エアライド Airride

ブーストとともに空中へ飛ぶ。
スニークの使用で爆破を起こして浮き続ける。
効果が切れるとそのまま落下する。

シールド : 一度きり
クールダウン : 10秒
効果継続時間 : 4秒

バステト Bastet

一時的に猫の神気をまとうことで超速を手に入れる。
特有のパーティクルを纏うのでバレやすい。
左クリックで一度だけ超加速できる

シールド : 一度きり
クールダウン : 9秒
効果継続時間 : 6秒

ヘイズ Haze

視界妨害をするエレメントを投げる。
内側からは完全に視界が妨害される。
効果を受けると、ウェポンの使用ができなくなる。
およそ5秒間このヘイズ状態が続く。

シールド : 一度きり
クールダウン : 15秒
効果継続時間 : 10秒

シールド Shield

弓矢を一度だけ受けれるようにするシールドを張る。
キルすることで張れるシールドが増加していく。
なお、重複させてシールドはつけられない

シールド : 1 + キル数分
クールダウン : 0秒

リーパー Leaper

爆発的な跳躍を発動させる。
常設してあるトランポリンでのジャンプ力を容易に超える。

シールド : 一度きり
クールダウン : 10秒

ショックウェイバー ShockWaver

爆風を作るエレメントを投げる。
近づくと強力なノックバックをくらう。

シールド : 一度きり
クールダウン : 15秒
効果継続時間 : 10秒

パラレル Parallel

立ち位置を記憶し、過去の記憶と入れ替えをする。
その際に過去の記憶先にテレポートさせる。
スニークしながら使用で位置の再設定が可能だが、全プレイヤーに密告サウンドが再生される。

シールド : 一度きり
クールダウン : 16秒

ヴァニッシャー Vanisher

一定時間透明になることができる。
これこそ逃げと奇襲に徹した代物。
なお透明になれるが、当たり判定は存在する
透明化の際に、全プレイヤーに向けて、警告サインが送信される。

シールド : 一度きり
クールダウン : 9秒
効果継続時間 : 6秒

スワッパー Swapper

自分と視線先の敵との位置を交換する。
自分は交換相手の向いている方向を向きながら交換され、相手は視点がランダムに搔きまわされる。

シールド : 一度きり
クールダウン : 13秒

ドーピング Doping

禁忌行為、ドーピングをする。
一時的に速度と跳躍力が上昇する。
上空にいるときに左クリックで、視点先に微妙なブーストが可能。

シールド : 一度きり
クールダウン : 8秒
効果継続時間 : 7秒


サプライアブルウェポン

ビーム Beam

Mob研究で得られた産物。武器直線軌道に矢を5つ放てる。
2tick置き(0.1s)に発射するため、微妙な方向調整も可能。
サプライアブルウェポンのひとつ。

シールド : なし
クールダウン : 20秒

グラヴィティ Gravity

プレイヤー全ての重力を一瞬増強する。
浮遊しているプレイヤーを落下させられる。
全プレイヤーに10秒間、めまいを付与。
サプライアブルウェポンのひとつ。

シールド : なし
クールダウン : 10秒

バトルアックス BattleAxe

近接最強のウェポン
攻撃無効時間もニンジャより極めて短い。
弓矢攻撃のほかに、打撃での攻撃が可能になる。
サプライアブルウェポンのひとつ。

攻撃クールダウン : 0.5秒
