初期ウェポン

これらのウェポンはすでに開放してあるデフォルトのウェポンです

エアライド ❝Airride❞

ブーストとともに空中へ飛ぶ。
スニークの使用で爆破を起こして浮き続ける。
効果が切れるとそのまま落下する。

鎧 : 1つ
クールダウン : 12秒
効果継続時間 : 4秒

もっと詳しく
 

バステト ❝Bastet❞

猫の神気をまとうことで一時的に超速を手に入れる。
特有のパーティクルを纏うのでバレやすい。
左クリックで一度だけ超加速できる。

鎧 : 1つ
クールダウン : 9秒
効果継続時間 : 6秒

もっと詳しく
 

リーパー ❝Leaper❞

爆発的な跳躍を発動させる。
常設してあるトランポリンでのジャンプ力を容易に超える。
ウェポンを持ちながらトランポリンを使うと通常より高く飛べ、ブロックを3秒間スライムブロックに変える。

鎧 : 1つ
クールダウン : 9秒

もっと詳しく
 


購入ウェポン

これらのウェポンは購入することで新規に使えるようになるウェポンです

ドーピング ❝Doping❞

禁忌行為、ドーピングをすることで一時的に速度と跳躍力が上昇する。
Hazeのデバフ効果を受けない唯一のウェポン。
上空にいるときに左クリックで、視点先に微妙なブーストが可能。
ウェポンを持ちながらムーブメントを使うと通常より長く効果を得られる。

鎧 : 1つ
クールダウン : 10秒
効果継続時間 : 7秒

もっと詳しく
 

ヘイズ ❝Haze❞

視界妨害と強力なデバフを有したエレメントを投げる。
内側からは完全に視界が遮断される。
デバフ効果を受けるとウェポンの使用ができなくなり、鈍足と吐き気を喰らう。
5秒間このヘイズ状態が続く。

鎧 : 1つ
クールダウン : 15秒
効果継続時間 : 10秒

もっと詳しく
 

パラレル ❝Parallel❞

立ち位置を記憶し、過去の記憶と入れ替えをする。
その際に過去の記憶先にテレポートさせる。
使用すると全プレイヤーに密告サウンドが再生される。
奈落の上で使用をした際、次のテレポート先が履歴からランダムの設置先に変化する。

鎧 : 1つ
クールダウン : 14秒

もっと詳しく
 

シールド ❝Shield❞

弓矢を一度だけ受けれるようにする鎧を新たに生み出せる。
キルすることで2つまで鎧を獲得できる。
なお、重複させて鎧はつけられない。
金ブロックの上で使い、10秒間留まることで一度だけ鎧を獲得できる。

鎧 : 1 + キル最大 2 + 最大獲得数 1
クールダウン : 最大12秒

もっと詳しく
 

ショックウェイバー ❝ShockWaver❞

デバフ効果を有し、衝撃波を作るエレメントを投げる。
近づいた全てのエンティティに強力なノックバックをくらわせる。
敵が当たると受ける鈍足のデバフは、全ウェポントップ。
自分もノックバックは、くらうので注意。

鎧 : 1つ
クールダウン : 12秒
効果継続時間 : 10秒

もっと詳しく
 

スワッパー ❝Swapper❞

自分と視線先の敵との位置を交換する。
自分は交換相手の向いている方向を向きながら交換され、相手は視点がランダムに搔きまわされる。
位置取りを覆せ。

鎧 : 1つ
クールダウン : 13秒

もっと詳しく
 

ヴァニッシャー ❝Vanisher❞

一定時間透明になることができる。
これこそ逃げと奇襲に徹した代物。
なお透明になれるが、当たり判定は存在する。
透明化の際に、全プレイヤーに向けて、警告サインが送信される。

鎧 : 1つ
クールダウン : 13秒
効果継続時間 : 6秒

もっと詳しく
 

トラッパー ❝Trapper❞

部分的に破壊可能な色ガラスの囲いを展開可能になる。
設置時の視点向きで囲いの形状が決まる。
敵も自分もトラップにかかる。
起動した罠は15秒後に自動消滅する。
罠に敵がかかると、強力な鈍足が付与される

鎧 : 1つ
クールダウン : 15秒
効果継続時間 : 15秒

もっと詳しく

高級ウェポン

これらのウェポンは商売人から受け取れる、強化型のウェポンです。

バトルアックス ❝BattleAxe❞

近接最強のウェポン
打撃での攻撃が可能になる。

攻撃クールダウン : 1秒

もっと詳しく
 

ビーム ❝Beam❞

射撃最強のウェポン
直線軌道に超速の矢を5つ放てる。
2tick置き(0.1s)に発射するため、微妙な方向調整も可能。

クールダウン : 20秒

もっと詳しく
 

グラヴィティ ❝Gravity❞

特殊最強のウェポン
プレイヤーの重力を増強する。
浮遊しているプレイヤーを落下させる。
全プレイヤーに7秒間、めまいを付与。
全プレイヤーに4秒間、鈍足を付与。

クールダウン : 14秒

もっと詳しく
 

ニンジャ ❝Ninja❞

移動最強のウェポン
忍者の能力を得ることで、壁走りと水上歩行が可能になる。
さらに、天井にも張り付くことができるようになる。
バステトと同様の速度が付与され、三回までなら空を蹴ってジャンプが可能。

クールダウン : 25秒
効果継続時間 : 10秒

もっと詳しく
 

スマッシャー ❝Smasher❞

地形破壊を行う爆破物を投げる。
広範囲のエリアが爆破されるため、奈落を生み出すことにも使える。

クールダウン : 25秒
効果継続時間 : 10秒
