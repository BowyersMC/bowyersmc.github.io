Chapter-02

私の仕事はこれが最期だ。
仲良しごっこはやめにしよう。もう今から創造主として牙をむく時である。

全ての「イミテイター」を作った科学者がプロファイルを脳にインプットさせて、アリーナへ参戦する。
対象から因子を抽出し、それをもちいて疑似的な「イミテーション」を引き起こしながら敵イミテイターを殲滅する匿名性をさらに上書きした性能を持つ。
対象の因子を奪う前が弱くなると判断した彼は、自身に驚異的な脚力を施し、壁ジャンプの威力を倍増させた。

イミテイター解説ページへ移動

姉妹鯖であるImitatorsMCのプロジェクト閉鎖を受けて、開発元は二者の共存を目指して能力の移植に取り組んできました。
今回はその第三種類目として、ネザー側の施設より送られたTrapperの兄であるVillainを受け継いだサプライウェポンの実装です。
Villainの扱う複数のVexは、壁を貫通しながら索敵をして敵を見つけ次第アタックを仕掛けて盲目を与えます。
近距離で使われるとすべてのVexが突撃してくるため、ほぼ半永久的に盲目状態が続く、驚異的な能力を持つ。

これに加えて、Doctorの実装とストーリーの関係性から金の鎧がサプライウェポンとして追加されました。

サプライ解説ページへ移動

各種イミテイターの調整

Airride / エアライド

  • クールタイムを14秒から18秒に変更。
  • 羽ばたきの横方向倍率を全盛期の1.1を超えた1.105に変更。
  • 羽ばたきの縦方向倍率を0.8から1.0に変更。

Bastet / バステト

  • クールタイムを11秒から15秒に変更。
  • 鈍足のデバフを削除。
  • 暗視のエフェクトを削除。
  • ブーストの最大倍率を3倍から4倍へ変更。

Canopy / キャノピー

  • クールタイムを27秒から20秒に変更。
  • 鎧なし時の特殊クールタイムを削除。
  • 着地時にエリトラが外れる仕様に変更。
  • 滑空時の射撃制限を削除。

Chorus / コーラス

  • クールタイムを12秒から11秒に変更。
  • 捕捉可能距離をSwapperと同じ70mに増加。
  • 周囲への喚き盲目を1秒から1.7秒に変更。

Doping / ドーピング

  • クールタイムを12秒から18秒に変更。

Haze / ヘイズ

  • 雪玉をマグマキューブが溶かす能力を追加。
  • マグマキューブの回転を削除。
  • クールタイムを16秒から12秒に変更。
  • 鈍足デバフのほかに盲目デバフを追加。
  • デバフのクールタイム加算値を7秒から10秒に変更。
  • 腕振りでマグマキューブを自分の座標側へ引き寄せる能力を追加。

Herobrine / ヘロブライン

  • エアストレイフの倍率を0.5から0.7に変更。
  • 捕捉失敗時のクールタイムを削除。
  • 凝視可能距離を60mから描画距離外の100mまで延長。

Leaper / リーパー

  • 使用状態によるクールタイムを削除。
  • クールタイムを一律10秒に変更。

Parallel / パラレル

  • 最大硬直時間を2.9秒から2秒へ変更。
  • クールタイムを14秒から13秒に変更。
  • 周囲への喚き盲目を1.9秒から2秒に変更。

Rider / ライダー

  • クールタイムを17秒から11秒に変更。
  • 発動失敗時のクールタイムを削除。
  • 降りたときのラヴェジャー残留時間を0.5秒から1秒に変更。
  • 衝突時対象のエフェクトを削除するように変更。

Shield / シールド

  • 破壊されたときの鎧自動装備を追加。
  • デフォルトのクールタイム仕様の削除。
  • チャージ時間を7秒から5秒へ変更。
  • 鎧のキル時獲得可能数の上限を削除。

ShockWaver / ショックウェーバー

  • クールタイムを13秒から8秒に変更。
  • デバフの鈍足の強さをⅣからⅤに変更。

Spear / スピアー

  • クールタイムを21秒から15秒に変更。
  • 効果継続時間を12秒から15秒に変更。
  • ダークネスを削除。
  • 鎧のない敵にあてたときの3秒の盲目付与能力を追加。

Steve / スティーブ

  • 持てるブロックを24から32に変更。
  • クールタイムを8秒から3秒に変更。

Trapper / トラッパー

  • クールタイムを15秒から13秒に変更。
  • 罠の有効時間を16秒から15秒に変更。
  • デバフのCT加算値を3秒から6秒に変更。
  • BAD_OMENのデバフを8秒から10秒に変更。
  • 罠発動時、対象に盲目を追加。
  • 透明化していても、罠を矢で当てると破壊できるように変更。
  • 設置したTrapperは罠にかからないように変更。
  • 設置失敗時のクールタイムを削除。

Twins / ツインズ

  • 鎧なし時の特殊クールタイムを削除。
  • クールタイムを15秒から10秒に変更。

Vanisher / ヴァニッシャー

  • 矢を構えたときの殺気の赤靄を削除。
  • クールタイムを12秒から13秒に変更。
  • 効果継続時間を6秒から8秒に変更。
  • 完全透明の仕様を透明化エフェクトへ変更。
  • 効果使用時、自動で鎧が外れるように変更。
  • 見られているときの鼓動を削除。
  • 矢を撃った時の透明化解除を削除。
  • Xrayによる透視を削除。

Xray / エクスレイ

  • クールタイムを14秒から15秒に変更。
  • 効果継続時間を7秒から8秒に変更。
  • 矢の重さを削除。
  • 強く引いた矢以外の貫通判定をランダム化。
  • 透過ブロックしか貫通矢が貫通しないように修正。(不可解な利用法が横行したため)

Yeti / イエティ

  • クールタイムを18秒から15秒に変更。
  • 効果継続時間を12秒から18秒に変更。
  • 雪玉の個数を7から8に変更。

追加要素

奈落落とし対策の強化

プレイヤーの周囲7ブロックに奈落があると、殴りノックバックが反応しなくなりました。

ペットの誇りの追加

超高額なすべてのペットを購入し終えると、新しい誇りが追加されるようになりました。
