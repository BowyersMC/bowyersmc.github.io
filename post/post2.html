09パッチの詳細説明

PHASE-09で更新される要素についてまとめました。

凝視殺人者の参戦

Minecraftの都市伝説の一つでもあるヘロブラインがBowyersMCへ参戦します。
全てのイミテイターから、エアストレイフの発動プロファイルを奪い去り、自らの物にした凶悪な「バグ」。
見つめ続けることで相手を一発で死へ追いやってしまう奇妙な能力をもった脅威的なイミテイター。

イミテイター解説ページへ移動

各種イミテイターの調整

Bastet / バステト

  • 超加速で発動座標を中心とした半径16ブロックのプレイヤーに音を発するようにしました。
  • クールタイムを9秒から、10秒にしました。
  • ブーストを使い終わった時に速度上昇レベルが低下する仕様を追加しました。

ShockWaver / ショックウェーバー

  • Meteorの隕石を吹き飛ばしていたのを修正しました。
  • Penetratorを吹き飛ばしていたのを修正しました。

Shield / シールド

  • 鎧獲得可能最大個数を2から3に変更しました。

Swapper / スワッパー

  • 水上や水中で使った時に、自分へ3秒の盲目がつくようにしました。

Vanisher / ヴァニッシャー

  • 鎧がない時に透明化時間を2秒増やす仕様を追加しました。
  • 鎧がない時のクールタイム減少を削除しました。

Trapper / トラッパー

  • 罠を壁ジャンプで抜けられないように専用のエフェクトを5秒間与えるようにしました。
  • 設置されたトラップをスライムに吹き飛ばされなくしました。

Spear / スピアー

  • 当たると移動系イミテイターのトリガーを消滅させる能力を、フグに追加しました。
  • 落っこちたフグが地面でさらに元気に跳ねるようになりました。
  • 発射されたフグの重力を大きくしました。

Yeti / イエティ

  • 雪玉の個数を5個から7個に変更しました。
  • 雪玉の当たり判定と速度を上昇させました。
  • 凍結を毎ヒット0.7秒に落とし、蓄積処理を修正しました。
  • 所持中なら常に水を凍らせられるようにしました。
  • クールタイムをSpearと同じにしました。

Twins / ツインズ

  • 敵に矢で双子が射抜かれると、クールタイムが0になる仕様を追加しました。
  • スニークの集合トリガーを、左クリックに変更し、集合でテレポートする方をデコイに変更しました。
  • 集合可能距離を20に変更しました。
  • 動かすか止めるかを定めるトリガーを削除しました。

Rider / ライダー

  • 左クリックトリガーに一度きりの特大ジャンプを追加しました。
  • 敵に衝突した時に、盲目を一瞬与える能力を追加しました。
  • トラップと巨大スライム以外の一般エンティティを吹き飛ばせるようにしました。
  • クールタイムを4秒増加しました。

Chorus / コーラス

  • 効果継続時間を13秒から15秒に変更しました。
  • 手招き可能距離を50から60に変更しました。

09の修正要素

一部モードの落下ログ修正

Rotation及び、KillStreakなどのリスポーン可能なゲームにおいて、一度敵イミテイターにキルされた後に奈落死すると、ログがおかしくなるのを修正しました。

Squadsのピン機能の修正

弓を持ちながらスニークと腕振りで、視点方向へピンを指す仕様が、上手く動いていない問題を修正しました。

梯子や蔦での反発を削除

梯子や蔦(ツタ)に掴まっているときにスニークすると、壁ジャンプが発動してしまう問題を修正しました。

水中での反発を削除

水中でスニークを行うと、なぜか壁ジャンプが発動してしまう問題を修正しました。

エントリーでの殴りKBを緩和

ひとによってはストレスに感じる人がいるため、KB抑制の鎧を渡すことで様子見します。

09の新規要素

ハイティアマントを追加

全ての所持イミテイターウェポンのティアで、ひとつでもSEVEN以上の物がある場合に買える、ハイティアマントを追加しました。

クールタイム終了の効果音を追加

クールタイムが終わった時に小さなサウンドを流すことで、ホットバーに目を当て続けなくても発動可能タイミングを図れるようにしました。

エアストレイフを削除

奇怪な動きをするエアストレイフを、新規イミテイター「Herobrine」の特権にしました。ウェポンの性能がかなり特殊で、対面の戦闘において即時効果を得られないため、この仕様に変えました。

練習場でのインベントリーにNinjaを追加

他のプレイヤーに影響を及ぼさないNinjaのみをインベントリーに入れるようにしました。

Gravityを一部モードからBAN

Rotation及び、KillStreakなどのリスポーン可能なゲームにおいて、リスポーンしたプレイヤーに対してあまりにも効力が強すぎたため、サプライプールから削除しました。

ペットの追加

1.20から実装された「スニッファー」をペットとして追加しました。

開発者のコメント

はじめに

今回のウェポン調整は主に「生成系」と「変化系」の2つのカテゴリーを大幅に強化しました。
はじめはBastetだけを調整するつもりでいましたが、全てのウェポンの平均の強さバランスが兼ねてより減少傾向にありました。
PHASE-01で猛威を振るったParallelに加え、PHASE-02で騒がれたVanisher、PHASE-04でリワークがされたAirride、PHASE-05のTwinsなど。
これらのナーフを行ったがゆえに、手を加えてこなかったBastetが相対的に強くなっていました。
加えて移動系イミテイターが圧倒的な強さを誇るのがPHASE-08になっても変わっていませんでした。そのため今回の調整は「移動系並みの性能を持たせるために」という目標を掲げて実装しました。

Herobrineとエアストレイフ

表示

新規イミテイターのHerobrineは全てのイミテイターと異なり、ウェポン能力を発動したときに即座に効果を得られません。これはShieldにもあてはまるのですが、Shieldの場合はチャージで鎧を獲得できるほかにキルすることで鎧が増えます。そもそもShieldは、キルした後を効果発揮の前提とした能力を備えており、一方でHerobrineは効果発動のために10秒間必要というリスクを負っています。Bastetのようにエフェクトが得られるわけでもなく、ただただ死のカウントを減らすだけで、敵からしたらただの無能力です。
その為、強さバランスの均衡を保つために異質なイミテイターである証としてHerobrineにのみ、ウェポン所持中のパッシブでエアストレイフを占有させました。
エアストレイフにクールタイムを設定して何度も連発をできないようにすることで、エアストレイフ自体の弱体化をしています。
死のカウントを進めるために、弾除けの要素としてエアストレイフを実装している形でありますが、あきらかなデバイス差が生まれていると判断した場合は、能力をブラッシュアップします。
その場合、エアストレイフは完全に全てのイミテイターから削除し、Ninjaに移植する形になります。


Bastetのナーフ理由

表示

Minecraftにおける対人ゲームにおいて、「速度」の変化というのは例えレベルが低くとも距離が近ければ近いほど、かなりな強さを敵に感じさせます。
そして、Bastetに限ってはエメラルドブロックを用いることで転移系を凌ぐ距離を「移動」できてしまうことが、かなりの強さを誇っています。
BastetをナーフしてDopingをナーフしなかった理由は、Bastetの持つ移動スキルがDopingと比にならないほど高いためです。
Leaperも上に行けるから転移系を凌ぐのではないかと思われがちですが、XZ軸上のあらゆる角度に移動できるBastetと上移動しかしないLeaperを並べてみたらそれはもうどちらが壊れているのかは明らかでしょう。
敵につめるときのBastetの戦闘を何度も見てきましたが、弓を構えるのはおろか、移動したことに気づけていない場面も数多くありました。これは超高速移動によりエフェクトパーティクルが完全に消えてしまい、気づけていないのが一番の原因であると考えました。
以上のことから、発動した瞬間やブーストの倍率には手を加えず、ウェポンのトリガーを使い終わったあとの状態に手を加えた調整をしました。


Swapperのナーフ理由

表示

転移系のイミテイターの特徴はなんといってもテレポートによる状態の立て直しの良さです。
Parallelは、瞬時に過去の座標へ移動する点と、奈落での使用による履歴シャッフルの能力が極めて強く、これに対して硬直のデバフを実装しました。
Chorusは、一度の使用で最高4つの座標を行き来できる点で強いですが、位置が明確であるという点で強さのバランスが取れています。
Swapperは、敵と自分の座標を入れ替える上に相手の視点がブレる点で強いですが、自分が危険に晒される可能性があるという点でバランスを取っています。
しかし、Swapperは1v1の時にこの弱点がほぼ完全に消えるのと、水上で使うことで一方的に撃ち下ろせる環境を生み出せます。
これでは強さのバランスを図れていないことが明らかにわかるため上項で記した弱体化をしました。


Twinsの1v1特化型強化

表示

Twinsはたとえ騙せたとしてもそれがなんだ？というような能力でした。
かつて起きたParallelの上位互換化を経て弱体化されたTwinsですが、結局のところ強みを発揮する機会があまりなく、扱いづらさだけが残ってしまいました。
今回イミテイターウェポンのコアのブラッシュアップをし、クールタイムの破棄を実装し、それをTwinsだけに特別に実装しました。
敵を欺くたびに、双子に変化し続け、欺ける限り無限に敵へ50%をぶつけ続ける凶悪イミテイターにしました。


Yetiの修正に伴う調整

表示

Yetiの能力に多くの欠陥が見つかったことから修正と強化を行い、ブラッシュアップという形で調整をしました。
雪玉の当てやすさをかなり上昇させ、実装に欠陥のあった「凍結時間の加算」をしっかりと適応しました。
さらにウェポン所持中に水を凍らせる能力を導入し、数少ないパッシブ持ちイミテイターに変更しました。
雪玉の数も5つから7つに増やし、それに伴って効果時間も伸ばしました。
YetiはSpearと同時に作ったイミテイターであり、親和性が高い扱い方でもあります。かなりの強化を得たのでクールタイムを相応の値へ引き上げました。
この先、Yetiには移動系キラーになってくれることを祈っています。


Spearのフグにまつわる強化

表示

参戦してから5フェーズもの間、鎧なしの敵に対してフグになんの価値もない「不遇なフグ」と化していました。
移動系イミテイターが効果を発動したその瞬間に、フグをぶん投げて当ててあげることで、穏やかに回る毒が、移動系イミテイターのトリガーを破壊します。
特に鎧を持った移動系イミテイターに大して破格な強さを誇ってしまうことが懸念点ではあるため、ナーフをPHASE-09の半ばで行う可能性があります。
